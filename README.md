# FCRN-PyTorch

## About the Project
Fully Convolutional Residual Network PyTorch implementation based on paper "Deeper Depth Prediction with Fully Convolutional Residual Networks" by Laina I. et al (https://arxiv.org/pdf/1606.00373.pdf).
